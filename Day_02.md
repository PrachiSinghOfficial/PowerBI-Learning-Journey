# 📖 Theory

## 1. Filters

Filters are used to display only the required data in a report.

Types of filters:

- Visual Level Filter
- Page Level Filter
- Report Level Filter

Filters help users analyze specific information without changing the original dataset.

---

## 2. Merge Tables

Merge Tables is a Power Query feature used to combine two tables based on a common column.

Example:

Customer Table + Orders Table

Common Column:

Customer ID

The result is a single table containing information from both tables.

---

## 3. Relationships

Relationships connect multiple tables inside the data model.

They allow Power BI to analyze data correctly.

Example:

Customers Table

↓

Customer ID

↓

Orders Table

---

## 4. One-to-Many Relationship

One record in one table is connected to multiple records in another table.

Example:

One Customer

↓

Many Orders

This is the most common relationship used in Power BI.
