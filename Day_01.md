# 📖 Theory

## 1. What is Power BI?

Power BI is a Business Intelligence (BI) tool developed by Microsoft.

It is used to:

- Analyze data
- Create interactive dashboards
- Build reports
- Visualize business insights
- Share reports with others

---

## 2. Power BI Desktop Interface

Main components of Power BI Desktop:

- Home Ribbon
- Report View
- Data View
- Model View
- Fields Pane
- Visualizations Pane
- Filters Pane

---

## 3. Importing Data

Power BI allows us to import data from different sources.

Examples:

- Excel
- CSV
- SQL Database
- Folder
- Web
- SharePoint

---

## 4. Power Query

Power Query is used to clean and transform data before creating reports.

Some common tasks include:

- Remove Columns
- Rename Columns
- Change Data Types
- Remove Duplicate Values
- Replace Values
- Filter Data

---

## 5. Data Transformation

Data transformation helps make raw data clean and ready for analysis.

Examples:

- Formatting dates
- Removing blank rows
- Correcting data types
- Splitting columns
- Merging columns

---

## 6. Parameters

Parameters allow users to create dynamic reports.

They can be used to:

- Change data source
- Filter data dynamically
- Improve report flexibility
